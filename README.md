****📌** Project Overview******

This project is a complete end-to-end AI system that automatically:

- 📧 Classifies incoming emails into categories
- ⚡ Detects urgency levels (High / Medium / Low)
- 📊 Provides real-time analytics through a dashboard

It is designed to simulate a real-world enterprise customer support system.


**🎯 Core Features**
📧 Email Classification

- Complaint
- Request
- Feedback
- Inquiry
- Spam
- Account



**⚡ Urgency Detection**

- High
- Medium
- Low


**📊 Smart Dashboard**

- Executive analytics
- Category distribution
- Urgency breakdown
- Email volume insights


**🎛️ Filters & Controls**

- Filter by category
- Filter by urgency
- Search functionality
- Date-based filtering


**🧠 Machine Learning Pipeline**

- Text preprocessing (cleaning, stopwords, normalization)
- TF-IDF Vectorization (3000 features)
- Models used:
  - Naive Bayes
  - Logistic Regression
  - Support Vector Machine (SVM)
- Hyperparameter tuning (GridSearchCV)
- Class-specific data augmentation
- Advanced embedding improvements
